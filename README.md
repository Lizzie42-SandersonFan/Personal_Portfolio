# Personal_Portfolio

## How to Use
Before running the program, you need to install tkinter. Type the folowing command into your terminal:
- pip install tkinter

After that is complete, go the the file called 'main.py' and hit the run button. A window will pop up with buttons you can press!

## Project Features
- Tkinter User Interface
- Buttons to navigate to different windows
- Buttons to run the featured programs
